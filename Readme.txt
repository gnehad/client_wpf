Je n'ai pas réussi à déployer mais j'ai trouvé une petite alternative pour tester l'affichage de la liste des employés 
dans un site Web : j'ai ouvert deux instances de Visual Studio, l'une pour le projet WPF et l'autre pour le site APS.NET Core.

L'application WPF ouvre une boite de dialogue dans laquelle copier l'URL du site puis elle lance le navigateur par défaut 
avec cette URL. Pour réaliser mes tests, je lance le projet ASP.NET dans le IIS intégré de Visual Studio et je copie son URL 
tout en le gardant ouvert.
